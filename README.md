# EventTicketing_CCITUBES

##Deskripsi 
project ini merupakan sistem backend untuk platform event ticketing dan booking
fitur utama : 
-Manajemen event(CRUD)
-Booking ticket menggunakan kuota
-pembuatan ticket
-ticket validation saat check-in
-role based access control (admin, organizer, customer)
---

##techstack
-**runtime** : node.js
-**framewokr** : express.js
-**ORM** : prisma
-**database** : PostgreSQL/MySQL
-**authentication** : JWT
-**password hashing** : bcrypt
-**validation** : zod / joi
-**file upload** : multer
-**API documentation** : swagger/OpenAPI
-**Testing**: Postman
